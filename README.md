# ✈️Airline Passenger Satisfaction Analysis

## 📌Project Overview
This project analyzes airline passenger satisfaction data to identify factors influencing customer satisfaction and travel experience. The analysis was performed using Excel for data cleaning and Power BI for interactive dashboard creation.

## Objectives
- Analyze passenger satisfaction levels
- Identify factors affecting customer experience
- Examine service quality ratings
- Analyze delay impacts on satisfaction
- Create interactive dashboards for business insights

## 📁Dataset Information
- Dataset: Airline Passenger Satisfaction Dataset
- Records: 129,880 Passenger Records
- Features: Demographics, Flight Details, Service Ratings, Satisfaction Status

## 📊Tools Used
- Microsoft Excel
- Power BI

## Dashboard Pages

### 1. Overall Dashboard
- Total Passengers

### 2. Customer Satisfaction Analysis
- Satisfaction by Customer Type
- Satisfaction by Travel Type

- Satisfaction by age group

### 3. Service Quality Analysis
- Service Ratings Comparison
- Service Performance Metrics

### 4. Flight Performance Analysis
- Departure Delay Analysis
- Arrival Delay Analysis
